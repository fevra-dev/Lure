attack chain: clickfix clipboard injection

Attackers present a fake CAPTCHA or "verify you are human" page. When the user clicks, JavaScript writes a malicious PowerShell or cmd command to the clipboard. The page instructs the user to press Win+R and Ctrl+V to execute it. 8% of all phishing attacks globally (ESET H1 2025).
