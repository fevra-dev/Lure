malicious oauth consent — state parameter email encoding

This demo simulates the detection logic locally. No real OAuth requests are made. The URLs shown are analyzed by the state parameter abuse detector.

threat: storm-2372 state parameter abuse

Attackers craft OAuth authorization URLs where the state parameter contains the victim's email address encoded in base64. When the victim authenticates, the OAuth callback delivers the token and the decoded email to the attacker's redirect_uri — a clean C2 exfiltration channel riding legitimate OAuth infrastructure.

PhishOps OAuthGuard intercepts these URLs in webRequest.onBeforeRequest and decodes the state parameter to check for embedded email addresses.
