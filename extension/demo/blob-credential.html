attack chain: html smuggling via blob: urls

Attackers deliver encoded payloads via legitimate-looking pages. JavaScript decodes the payload with atob(), wraps it in a Blob, and navigates to a blob: URL. The resulting credential harvest page:

  • Never appears in proxy logs (blob: URLs are ephemeral)
  • Bypasses Safe Browsing URL checks entirely
  • Has no origin, so CSP headers don't apply
